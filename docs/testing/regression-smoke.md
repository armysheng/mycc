# MyCC 回归冒烟清单

> 目的：提供 PR 和主干合并前的最小回归标准。

## PR 必跑（P0）

1. 登录成功（含 token 正常写入）。
2. 首次用户 onboarding 可完成。
3. 聊天发送主链路可用（请求成功并有回复）。
4. 技能页可加载、可搜索、可安装/卸载。

## main 必跑（P1）

1. PR 必跑项全通过。
2. 技能启用/禁用联动正常。
3. 历史会话可见并可打开。
4. 错误提示链路可见且不白屏。

- 执行人：
- 环境：
- 用例结果：PASS/FAIL/BLOCKED
- 证据：截图/trace/日志链接
