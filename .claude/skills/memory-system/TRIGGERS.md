# 触发词配置

## 触发词

| 操作 | 触发词示例 |
|------|-----------|
| recall | "回忆一下", "查查记忆", "我记得" |
| remember | "记一下", "记住这个", "先存起来" |
| consolidate | "以后都这样", "沉淀一下", "写到长期记忆" |
| forget | "忘掉", "清理记忆", "这个记错了" |
| snapshot | "生成快照", "整理记忆" |

## 集成到 AGENTS.md

在每次对话开始时：
1. 读取 .memory/snapshot.md
2. 检索相关记忆

对话结束时：
3. 更新快照
