## Overview

mock-interview 是一个沉浸式面试模拟工具，帮助你在真正走进面试间之前做好充分准备。它能根据你的目标岗位、目标公司和经验水平，生成高度针对性的面试题目，还原真实面试的节奏和压力感。

支持多种面试类型：技术面试(算法、系统设计、语言特性)、行为面试(STAR 法则、情景题)、案例面试(商业分析、产品设计)。每次回答后，AI 会从内容完整性、表达清晰度、逻辑结构等维度给出详细点评和评分，并提供具体的改进建议和参考答案。

面试记录会被保存，你可以回顾历次模拟的表现趋势，发现自己的薄弱环节并针对性提升。无论是应届生求职、社招跳槽，还是晋升答辩，都能通过反复练习建立信心。
