## Overview

data-analysis 是一个数据分析和可视化工具，帮助你从 CSV、Excel 等表格数据中快速提取洞察。支持数据清洗、缺失值处理、异常值检测等预处理操作，让原始数据变得可分析。

内置常用统计分析功能，包括描述性统计、分布分析、相关性检验、趋势识别和同比/环比计算。可以自动识别数据中的关键模式和异常点，并给出业务层面的解读建议。

支持生成多种图表（折线图、柱状图、散点图、热力图、饼图等）进行数据可视化。对于复杂分析需求，可以直接执行 Python 代码（pandas、numpy、matplotlib、seaborn 等），灵活应对各种数据场景。
