## Overview

周报生成技能让写周报不再是负担。它会自动汇总你本周的对话记录、完成的待办事项、项目进展和关键产出，智能生成一份内容充实的周报。你只需确认和微调，省去从零回忆和整理的麻烦。

支持多种主流周报格式：传统的"本周完成/下周计划/需要协助"三段式、OKR 进展汇报格式、项目维度的进展总结、以及自定义模板。系统会根据你选择的格式自动调整内容组织方式和详略程度。

周报数据来源于你在 MyCC 中的真实工作痕迹，确保内容准确可靠。生成后支持一键复制、导出为 Markdown 或纯文本。历史周报自动归档，方便季度总结和年度回顾时快速检索。
