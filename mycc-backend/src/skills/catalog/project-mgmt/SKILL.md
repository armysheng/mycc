## Overview

项目管理技能为你提供轻量但完整的项目管理能力。通过自然语言即可创建项目、添加任务、设定里程碑和追踪进度。支持同时管理多个项目，每个项目拥有独立的任务看板和进度视图。

任务支持多级拆分、优先级设置、状态流转（待办/进行中/已完成）和负责人分配。内置甘特图视角，直观展示任务时间线和依赖关系。里程碑功能帮你把控关键节点，确保项目按计划推进。

项目数据与待办管理、周报生成等技能深度联动。项目中的任务自动同步到待办列表，项目进展自动纳入周报内容。支持项目归档和模板复用，让你的项目管理经验可以沉淀和复制。
