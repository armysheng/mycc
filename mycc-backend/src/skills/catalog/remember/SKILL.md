## Overview

记忆笔记让你的 AI 助手拥有真正的长期记忆。你可以随时告诉它需要记住的信息——个人偏好、项目细节、常用配置、重要决策等，这些信息会被持久化存储并在后续对话中自动召回。

记忆内容按主题自动分类组织，支持模糊搜索和关联查询。你可以问"我之前说过什么关于某个话题的"，系统会从记忆库中检索相关信息并整理呈现。随着使用时间增长，AI 会越来越了解你的工作方式和偏好。

支持记忆的增删改查和批量管理。你可以查看所有已存储的记忆、修改过时的信息、删除不再需要的条目。记忆数据完全私有，仅你本人可访问，打造真正属于你的专属知识库。

## 示例

> 记住我的 API key 放在 .env 文件里
> 我之前说过什么关于项目架构的
> 记住我喜欢简洁的代码风格
