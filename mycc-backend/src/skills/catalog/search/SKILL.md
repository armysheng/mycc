## Overview

search 技能让 CC 具备实时联网搜索能力。当你需要查询最新信息、验证事实、或获取实时数据时，CC 会自动调用搜索引擎，抓取多个来源的结果，提取关键信息并整理成结构化的回答。

支持中英文混合搜索，自动判断最佳搜索语言和关键词组合。搜索结果会附带来源链接，方便你溯源验证。对于复杂问题，CC 会进行多轮搜索，交叉比对不同来源的信息，确保回答的准确性。

典型使用场景包括：查询最新技术动态、了解时事新闻、比较产品和方案、查找官方文档、获取天气/汇率等实时数据。
