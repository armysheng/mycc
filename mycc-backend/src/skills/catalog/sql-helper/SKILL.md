## Overview

sql-helper 让你用自然语言写 SQL。告诉它你想查什么数据，它会生成对应的 SQL 语句，支持 MySQL、PostgreSQL 和 SQLite 三种主流数据库的语法差异。如果你提供了表结构信息，生成的查询会更精准；没有也没关系，它会基于通用最佳实践来写。

查询优化是 sql-helper 的另一个强项。把慢查询丢给它，它会分析执行计划、识别全表扫描和不必要的嵌套查询，给出具体的优化方案。同时推荐合理的索引策略，解释为什么加这个索引、预期能带来多少提升。

对于复杂的多表关联、子查询嵌套和窗口函数场景，sql-helper 会一步步拆解逻辑，先帮你理清数据关系，再生成可靠的查询语句。每条生成的 SQL 都会附带简要说明，确保你不仅拿到了结果，还理解了背后的逻辑。
