## Overview

xlsx 技能让 CC 能够创建和编辑 Excel 电子表格。无论是简单的数据整理还是复杂的数据分析，CC 都能帮你搞定。支持公式计算、数据透视、条件格式、图表生成等 Excel 核心功能。

你可以用自然语言描述需求，CC 会自动创建合适的表格结构：设置表头、填充数据、添加公式、应用格式。对于已有的表格数据，CC 可以进行统计分析、趋势计算、异常值检测，并生成可视化图表。

适用场景广泛：财务报表、销售数据分析、项目进度跟踪、库存管理、调查问卷统计等。生成的 .xlsx 文件完全兼容 Microsoft Excel 和 WPS 表格，公式和格式完整保留。
