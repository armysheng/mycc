## Overview

小红书文案技能专为小红书平台内容创作设计，深度理解小红书的内容调性和爆款规律。自动生成符合平台风格的吸睛标题（含恰到好处的emoji）、结构化正文和精准话题标签，帮你快速产出高互动率的笔记内容。

支持多种热门品类的内容生成，包括美妆护肤、美食探店、旅行攻略、穿搭分享、数码科技、居家好物、健身运动、学习干货等。针对每个品类，技能会自动匹配对应的内容框架和表达方式，比如美食类会突出口感描述和推荐理由，旅行类会注重行程安排和拍照攻略。

生成的内容遵循小红书高赞笔记的常见结构：开头用一句话抓住注意力，正文分点叙述便于阅读，结尾引导互动。标签会结合内容主题和当前热门话题，帮助笔记获得更多曝光。
