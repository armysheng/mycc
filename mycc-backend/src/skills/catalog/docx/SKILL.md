## Overview

docx 技能让 CC 能够创建和编辑 Word 文档。从一段粗略的文字到一份排版精美的正式文档，CC 帮你完成从内容到格式的全部工作。支持标题层级、段落样式、页眉页脚、目录生成等专业排版功能。

你可以用自然语言描述文档需求，CC 会自动选择合适的文档结构和格式。对于已有文档，支持内容修改、格式调整、添加批注、修订跟踪等编辑操作。还能根据模板批量生成格式统一的系列文档。

适用于各类办公文档：项目报告、技术方案、会议纪要、合同模板、规范文档等。生成的 .docx 文件完全兼容 Microsoft Word 和 WPS 文字，格式和样式完整保留，打印效果专业。
