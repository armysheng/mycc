## Overview

code 技能是 CC 的核心编程助手能力。无论你需要从零编写代码、理解现有代码的逻辑、还是排查棘手的 bug，CC 都能提供专业的帮助。支持 Python、JavaScript/TypeScript、Go、Rust、Java、C++ 等主流编程语言。

在写代码时，CC 会遵循对应语言的最佳实践和代码规范，提供可直接运行的完整代码，而不是片段。在解释代码时，会逐层拆解逻辑，标注关键设计决策。在调试时，会分析错误信息、定位根因并给出修复方案。

除了基础的读写代码，还支持代码审查、性能优化建议、架构设计讨论、技术选型对比等进阶场景。CC 会结合项目上下文给出最适合当前场景的建议，而不是泛泛的教科书回答。
