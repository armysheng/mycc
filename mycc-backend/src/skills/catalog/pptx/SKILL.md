## Overview

pptx 技能让 CC 能够创建和编辑 PowerPoint 演示文稿。你可以从一份大纲、一段文字、甚至一个想法出发，CC 会帮你生成结构完整、排版专业的 PPT 文件。也可以对已有的 PPT 进行修改和优化。

支持的操作包括：从大纲自动生成完整 PPT、添加/删除/重排幻灯片、修改文本和格式、插入图表和表格、调整布局和配色方案。CC 会根据内容类型自动选择合适的版式，确保视觉效果专业。

适用于工作汇报、项目提案、技术分享、产品演示等场景。你只需要专注于内容本身，排版和格式交给 CC。生成的 .pptx 文件完全兼容 Microsoft PowerPoint 和 WPS，可以直接使用或继续编辑。
