## Overview

pdf 技能用于处理 PDF 文件。你可以上传或指定 PDF 文件路径，CC 会解析文件内容，提取其中的文本、表格、图片描述等结构化数据，并根据你的需求进行进一步处理。

支持的操作包括：全文提取、智能摘要生成、关键信息抽取、表格数据提取为结构化格式、多 PDF 内容对比分析。对于长文档，CC 会自动分段处理，确保不遗漏重要内容。

适用于合同审阅、论文阅读、报告分析、发票信息提取等场景。无论是几页的简短文档还是上百页的长报告，都能快速定位你关心的内容。
