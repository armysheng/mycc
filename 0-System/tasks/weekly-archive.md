# 周记归档

**触发时间**：每周日 22:30

## 任务内容

1. 回顾本周 `status.md` 的所有记录
2. 汇总到 `0-System/context.md`
3. 归档到 `5-Archive/周记/第X周/`

## 模板

```markdown
# 第 X 周周记 - {{DATE_RANGE}}

## 本周概览
（本周关键词、主要成就）

## 每日回顾
（从 status.md 提取关键进展）

## 本周收获
（认知沉淀、新学到的）

## 下周计划
（重点方向）
```

## 执行方式

每周日 22:30 提醒 wannago 手动归档，或通过 mycc 的 scheduler 系统自动提醒。
